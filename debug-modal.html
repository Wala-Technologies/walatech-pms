Follow these steps to identify the persistent modal dialog:

Step 1: Open Browser Developer Tools

Press F12 or right-click and select "Inspect Element"

Step 2: Clear All Storage

In Developer Tools:

  1. Go to Application tab
  2. Under Storage, click "Clear storage"
  3. Check all boxes and click "Clear site data"

Step 3: Find Modal Elements

In the Console tab, paste and run this code:

// Find all modal elements const modals = document.querySelectorAll('[class*="modal"], [class*="Modal"], .ant-modal, .ant-modal-wrap'); console.log('Found modals:', modals); modals.forEach((modal, index) => { console.log(`Modal ${index}:`, modal); console.log('Visible:', window.getComputedStyle(modal).display !== 'none'); console.log('Classes:', modal.className); }); // Find elements with modal-related text const allElements = document.querySelectorAll('*'); const modalTexts = []; allElements.forEach(el => { if (el.textContent && el.textContent.toLowerCase().includes('modal')) { modalTexts.push(el); } }); console.log('Elements with modal text:', modalTexts);

Step 4: Check for Ant Design Modals

Run this code to find Ant Design specific modals:

// Check for Ant Design modals specifically const antModals = document.querySelectorAll('.ant-modal-root, .ant-modal-wrap, .ant-modal'); console.log('Ant Design modals:', antModals); // Check for modal masks const masks = document.querySelectorAll('.ant-modal-mask'); console.log('Modal masks:', masks); // Check for any visible overlays const overlays = document.querySelectorAll('[style*="z-index"]'); console.log('High z-index elements:', overlays);
